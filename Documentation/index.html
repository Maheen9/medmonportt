template installation


Follow the steps below to setup your site template:


  • 1. Unzip the HTML folder to find all the template files.
  • 2. Upload these files to your hosting web server using FTP or localhost in order to use it on your website.
  • 3. Upload all or specific HTML files as per your need.
  • 4. Make changes as per your requirements.
